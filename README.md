# 🎓 Sistema de Calificaciones (Streamlit App)

Aplicación web interactiva desarrollada con **Streamlit** para la consulta, análisis y visualización de calificaciones académicas a partir de un archivo Excel.

Permite explorar información a nivel de alumno, materia y periodo, incluyendo métricas clave y gráficos dinámicos.

---

## 📌 Características principales

- 📊 Consulta de calificaciones por alumno
- 📈 Estadísticos generales por periodo
- 📉 Análisis histórico por materia
- 📜 Historial académico (kardex) del alumno
- 🎨 Interfaz visual moderna con métricas y gráficos interactivos
- 📂 Carga dinámica de archivos Excel

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
|----------|-----|
| `streamlit` | Interfaz web interactiva |
| `pandas` | Manipulación de datos |
| `plotly` | Visualización de datos |
| `openpyxl` | Lectura de archivos Excel |
